# Syarat & Ketentuan

*Terakhir diperbarui: Mei 2025*

Dengan menggunakan JakInfra dan API-nya, kamu dianggap telah membaca dan menyetujui syarat berikut. Tolong baca dengan santai — tidak ada jebakan di sini. 😄

---

## 🔑 Penggunaan API

- **API Key bersifat personal** — satu key untuk satu proyek
- Dilarang menjual atau mendistribusikan ulang API Key ke pihak lain
- Dilarang menggunakan API untuk tujuan ilegal atau merugikan
- Ada rate limit wajar **(100 req/menit)** untuk menjaga kestabilan layanan

> Butuh limit lebih tinggi? Ajukan via [GitHub Issues](https://github.com/Ghifariezra/jakinfra/issues) dengan menyertakan nama proyek dan alasan penggunaan — kami akan review manual.

---

## 🗄️ Penggunaan Data

Data infrastruktur yang tersedia di JakInfra bersumber dari data publik Pemerintah DKI Jakarta, yang kemudian diolah dan distrukturisasi oleh tim JakInfra.

**Kamu boleh menggunakan data ini untuk:**

- Proyek personal atau riset akademik
- Aplikasi komersial — dengan mencantumkan kredit JakInfra

> **Kamu tidak boleh:**
> - Mengklaim data ini sebagai milik kamu sendiri
> - Mendistribusikan ulang data tanpa mencantumkan kredit

---

## ⚡ Ketersediaan Layanan

JakInfra adalah proyek open source yang dikelola secara **sukarela**.

Kami tidak menjamin uptime 100% dan berhak menghentikan atau mengubah layanan sewaktu-waktu. Kalau ada downtime, kami akan usahakan untuk info via GitHub.

---

## 📝 Perubahan Syarat

Syarat ini dapat berubah sewaktu-waktu. Penggunaan berkelanjutan setelah perubahan dianggap sebagai persetujuan terhadap syarat yang baru.
