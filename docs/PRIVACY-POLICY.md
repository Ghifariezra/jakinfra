# Kebijakan Privasi

*Terakhir diperbarui: Mei 2025*

JakInfra adalah proyek open source yang menyajikan data infrastruktur 
publik DKI Jakarta. Kami menghargai privasi kamu.

---

## 🗂️ Data yang Kami Kumpulkan

Saat kamu mendaftar untuk mendapatkan API Key, kami menyimpan:

- **Nama developer** — identitas kamu sebagai pengguna API
- **Nama proyek** — untuk memahami konteks penggunaan
- **Alamat email** — sebagai sarana komunikasi jika diperlukan
- **Tujuan penggunaan** — untuk memahami bagaimana data dimanfaatkan

> Kami **tidak** menyimpan password, data kartu kredit, atau informasi pembayaran apapun.

---

## 🔍 Bagaimana Data Digunakan

Data yang kamu berikan hanya digunakan untuk:

- Mengidentifikasi penggunaan API
- Menghubungi kamu jika ada perubahan layanan atau pelanggaran ketentuan
- Mencegah penyalahgunaan melalui rate limiting

> Kami **tidak** menjual atau membagikan data kamu ke pihak ketiga manapun.

---

## 🍪 Cookies & Tracking

JakInfra **tidak** menggunakan cookies tracking atau layanan analitik pihak ketiga seperti Google Analytics.

Satu-satunya data yang disimpan di browser kamu adalah preferensi tema *(dark/light mode)* via `localStorage` — dan itu sepenuhnya di sisi kamu.

---

## 🔐 Keamanan API Key

API Key yang kamu terima bersifat **rahasia dan personal**.

> Jangan bagikan API Key kamu ke publik atau commit ke dalam repository. Jika terjadi kebocoran, segera revoke melalui endpoint `/auth/keys/revoke`.
