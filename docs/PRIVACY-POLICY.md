# Kebijakan Privasi

*Terakhir diperbarui: Mei 2025*

JakInfra adalah proyek open source yang menyajikan data infrastruktur 
publik DKI Jakarta. Kami menghargai privasi kamu.

## Data yang Kami Kumpulkan

Saat kamu mendaftar untuk mendapatkan API Key, kami menyimpan:
- Nama developer atau nama kamu
- Nama proyek yang kamu bangun
- Alamat email — digunakan sebagai identitas dan sarana komunikasi
- Tujuan penggunaan — untuk memahami bagaimana data dimanfaatkan

Kami **tidak** menyimpan password atau informasi pembayaran.

## Bagaimana Data Digunakan

Data yang kamu berikan hanya digunakan untuk:
- Mengidentifikasi penggunaan API
- Menghubungi kamu jika ada perubahan layanan atau pelanggaran ketentuan
- Mencegah penyalahgunaan (rate limiting)

Kami **tidak** menjual atau membagikan data kamu ke pihak ketiga manapun.

## Cookies & Tracking

JakInfra tidak menggunakan cookies tracking atau layanan analitik 
pihak ketiga (seperti Google Analytics).

Satu-satunya data yang disimpan di browser kamu adalah preferensi 
tema (dark/light mode) via localStorage.

## Keamanan

API Key yang kamu terima bersifat rahasia. Jangan bagikan ke publik 
atau commit ke repository kamu.
