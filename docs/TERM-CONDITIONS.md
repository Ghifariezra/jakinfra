# Syarat & Ketentuan

*Terakhir diperbarui: Mei 2025*

Dengan menggunakan JakInfra dan API-nya, kamu setuju dengan 
syarat berikut.

## Penggunaan API

- API Key bersifat personal — satu key untuk satu proyek
- Dilarang menjual atau mendistribusikan ulang API Key
- Dilarang menggunakan untuk tujuan ilegal
- Ada rate limit wajar untuk menjaga kestabilan layanan
- Butuh limit lebih tinggi? Hubungi via GitHub Issues

## Data

Data infrastruktur yang tersedia di JakInfra bersumber dari 
data publik Pemerintah DKI Jakarta, yang kemudian diolah oleh tim JakInfra.

Kamu boleh menggunakan data ini untuk:
- Proyek personal atau riset
- Aplikasi komersial — dengan mencantumkan kredit JakInfra

Kamu **tidak boleh**:
- Mengklaim data ini sebagai milik kamu sendiri
- Mendistribusikan ulang data tanpa kredit

## Ketersediaan Layanan

JakInfra adalah proyek open source yang dikelola secara sukarela. 
Kami tidak menjamin uptime 100% dan berhak menghentikan layanan 
sewaktu-waktu tanpa pemberitahuan.

## Perubahan Syarat

Syarat ini dapat berubah sewaktu-waktu. Penggunaan berkelanjutan 
setelah perubahan dianggap sebagai persetujuan.
